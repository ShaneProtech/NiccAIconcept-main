Integration Code

To add this widget to your website, simply include this script tag:

<script src="http://localhost:5001/embed.js" data-auto-init></script>

Custom Configuration

You can customize the widget appearance and behavior:

<script src="http://localhost:5001/embed.js"></script> <script> NICCWidget.init({ position: 'bottom-left', buttonText: 'Vehicle Help', buttonColor: '#ff6b35', width: '450px', height: '650px' }); </script>
